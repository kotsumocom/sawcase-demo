# Badge

ステータスやカウントを示す小さなラベルコンポーネントです。

## 使い方

```html
<span class="st-badge st-badge--primary">新着</span>
```

## カラーバリアント

```html
<span class="st-badge st-badge--primary">Primary</span>
<span class="st-badge st-badge--secondary">Secondary</span>
<span class="st-badge st-badge--tertiary">Tertiary</span>
<span class="st-badge st-badge--error">Error</span>
```

## スタイルバリアント

### Outline

枠線のみのバッジです。

```html
<span class="st-badge st-badge--outline">Outline</span>
```

### Dot

テキストなしの小さな丸いインジケーターです。

```html
<span class="st-badge st-badge--dot st-badge--error"></span>
```

## サイズ

```html
<span class="st-badge st-badge--primary st-badge--sm">Small</span>
<span class="st-badge st-badge--primary">Default</span>
<span class="st-badge st-badge--primary st-badge--lg">Large</span>
```

## クラス一覧

| クラス | 説明 |
|--------|------|
| `st-badge` | ベースクラス（必須） |
| `st-badge--primary` | プライマリカラー |
| `st-badge--secondary` | セカンダリカラー |
| `st-badge--tertiary` | ターシャリカラー |
| `st-badge--error` | エラーカラー |
| `st-badge--outline` | 枠線のみスタイル |
| `st-badge--dot` | ドットインジケーター |
| `st-badge--sm` | 小サイズ |
| `st-badge--lg` | 大サイズ |
