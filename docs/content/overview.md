# Sawcase とは

**Sawcase** は Deno Fresh 専用の SaaS スターターキットです。

管理画面、ドキュメントサイト、認証画面、ランディングページなど、SaaS に必要な「骨格」を Preact コンポーネントとして提供します。

## 特徴

- **Fresh 専用**: Deno Fresh に最適化。`deno add` だけで即使える
- **コンポジション**: sawtooth-css のコンポーネントと自由に組み合わせ可能
- **レスポンシブ**: モバイルファースト設計
- **AI フレンドリー**: AGENT.md で AI アシスタントがテンプレートからページを生成可能

## 提供レイアウト

| プレフィックス | カテゴリ | 説明 |
|-------------|---------|------|
| `sc-admin-*` | 管理画面 | ヘッダー + サイドナビ + ページ構造 |
| `sc-docs-*` | ドキュメント | 3カラムレイアウト + TOC |
| `sc-auth` | 認証 | 中央配置カード |
| `sc-lp-*` | LP | ヒーロー + フィーチャー + フッター |

## クイックスタート

```bash
deno add @kotsumo/sawcase
```

sawtooth-css、@deno/gfm、Zag.js、lucide-preact は依存として自動インストールされます。

詳しくは [インストール手順](/docs/install) を参照してください。
